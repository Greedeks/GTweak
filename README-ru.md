<h2> Широкий спектр функций, включая: 🔩</h2>

```java
- Активация Windows с использованием методов HWID и KMS;
- Отключение Windows Defender, SmartScreen, Antimalware, VBS и контроля учетных записей (UAC);
- Отключение и приостановку обновлений Windows, а также удаление временных файлов обновлений;
- Отключение неиспользуемых и ненужных системных служб;
- Отключение кейлоггеров и телеметрии для Windows, NVIDIA и Intel;
- Отключение служб и событий, связанных со сбором диагностических данных пользователя;
- Отключение задач по сбору данных в планировщике задач Windows;
- Отключение ненужных сетевых протоколов: Teredo, ISATAP и IPv6;
- Отключение встроенных средств диагностики Windows и дефрагментации дисков;
- Отключение персонализированной рекламы и системных баннеров, включая экран SCOOBE;
- Отключение системных уведомлений, рекомендаций и советов Windows;
- Блокировку «теневых» доменов Microsoft, собирающих данные, через hosts и брандмауэр;
- Удаление OneDrive и Microsoft Edge вместе с очисткой всех связанных данных и компонента WebView2;
- Удаление стандартных предустановленных приложений (UWP) в Windows 10/11;
- Удаление и отключение ИИ-помощников: Cortana, Copilot и Recall;
- Настройку интерфейса: смену тем, параметров окон, расположения панели задач и иконок;
- Настройку клавиатуры и мыши: отключение фильтрации ввода, залипания клавиш и акселерации курсора;
- Оптимизацию настроек Windows и применение схемы питания «Максимальная производительность»;
- Настройку электропитания драйверов Realtek High Definition Audio для устранения задержки звука;
- Просмотр конфигурации оборудования и мониторинг системных компонентов;
- Сжатие и восстановление файлов и папок с помощью функций NTFS для экономии места;
- Очистку оперативной памяти, временных файлов, кэша и безопасное удаление папки Windows.old;
- Выполнение пользовательских скриптов (.ps1, .cmd, .bat, .reg) напрямую через утилиту с правами TrustedInstaller.
```

<h2> Системные Требования</h2>

> ⚠ Утилита предназначена только для официальных образов Windows, загруженные из надежных источников. Если у вас установлена обрезанная/измененная версия Windows, за работоспособность отвечаете вы сами.

```c++
Поддержка Windows: начиная с Windows 10 (build 18362.116)
Установленная платформа: .NET Framework 4.8
```
</br>

> [!WARNING]  
> Перед использованием остановите антивирус. Если вы используете Windows Defender, дополнительно добавьте её в исключения.
