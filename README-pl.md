<h2> Szeroki zakres funkcji, w tym: 🔩</h2>

```java
- Aktywacja systemu Windows przy użyciu metod HWID i KMS;
- Wyłączanie Windows Defender, SmartScreen, Antimalware, VBS oraz UAC;
- Wyłączanie aktualizacji Windows i usuwanie tymczasowych plików aktualizacji;
- Wyłączanie nieużywanych i zbędnych usług systemowych;
- Wyłączanie keyloggerów oraz telemetrii Windows i NVIDIA;
- Wyłączanie usług i zdarzeń związanych ze zbieraniem danych użytkownika;
- Wyłączanie zadań w Harmonogramie Zadań Windows;
- Wyłączanie niepotrzebnych protokołów sieciowych: Teredo, ISATAP i IPv6;
- Wyłączanie wbudowanych narzędzi diagnostycznych Windows oraz defragmentacji;
- Wyłączanie reklam i banerów systemowych w Windows (w tym SCOOBE);
- Wyłączanie powiadomień, rekomendacji i sugestii systemowych;
- Blokowanie ukrytych domen Microsoft zbierających dane;
- Odinstalowywanie OneDrive i Microsoft Edge wraz z usunięciem wszystkich powiązanych danych i folderów;
- Usuwanie standardowych, preinstalowanych aplikacji Windows 10/11;
- Usuwanie i wyłączanie asystentów AI: Cortana, Copilot oraz Recall;
- Dostosowywanie interfejsu: motywy, ustawienia okien i ikony;
- Konfiguracja klawiatury i myszy: wyłączanie filtrów klawiszy, klawiszy trwałych oraz akceleracji;
- Dostosowywanie ustawień Windows i modyfikacja planów zasilania;
- Regulacja ustawień zasilania sterowników Realtek High Definition Audio (naprawa opóźnień dźwięku);
- Podgląd konfiguracji i monitorowanie komponentów systemu;
- Implementacja kompresji danych NTFS z możliwością późniejszego przywrócenia.
```
